# rEach podcast

This repository contains an OKF-style knowledge bundle for the podcast series
**Die Laternenhüter des Stillen Tals**.

## Layout

- `OKF/` — the OKF source bundle
- `images/` — cover art and series artwork used by the notes
- `okf.yaml` — repository manifest for publishing workflows

## Publishing notes

The content is written as standard Markdown with YAML frontmatter so it can be:

- versioned in GitHub,
- rendered later with Quartz as HTML pages,
- and still read directly as OKF source.
